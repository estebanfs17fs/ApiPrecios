Api precios

Este proyecto se encarga de realizar un scrapper de los supermercados mas populares del Uruguay.

La idea es buscar productos, precios. Para que se puedan consultar un producto, su precio especifo en
un determinado momento.

Los datos seran guardados en una base de datos.

La api proporcionara una forma de consulta por otras aplicaciones.

Los datos seran actualizados de forma periodica por la aplicacion, ante ejecuciones programadas
por el administrador.
